# Changelog

이 프로젝트의 모든 주목할 만한 변경 사항은 이 파일에 기록됩니다.

## [1.0.1] - 2026-04-16

### Security (보안)
- **RLS(Row-Level Security) 강화**: 모든 주요 테이블에 행 레벨 보안을 활성화하여 비인가 접근 차단.
- **관리자 권한 체계 구축**: `is_admin()` SQL 헬퍼 함수를 통한 엄격한 접근 제어 적용.
- **익명 주문 허용**: 보안을 유지하면서 정식 서비스 요건인 로그인 없는 주문 기능 지원.

## [1.0.0] - 2026-04-12

### Added (추가)
- **배달 주문 시스템**: 외부 제휴 업체 연동 기능 (주문 생성, 메뉴 조회, 관리자 알림).
- **배달 트래커 위젯**: 실시간 배달 상태 추적 및 역동적인 상태 변경 애니메이션 추가.
- **은하수(Milky Way) 배경**: 프리미엄 애니메이션이 적용된 우주 테마 배경 도입.
- **실시간 알림 통합**: 매장 음식 주문 및 외부 배달 주문에 대한 통합 알림 지원.
- **버전 관리 체계**: Semantic Versioning 도입 및 UI 내 버전 표시 기능.

### Fixed (수정)
- **실시간 구독 누락**: ClientLauncher에서 배달 주문의 상태 변경을 감지하지 못하던 문제 해결.
- **배포 경로 최적화**: GitHub Pages 서브디렉토리 배포를 위한 Vite base 경로 정규화 및 리소스 로드 문제 해결.
- **어드민 스크립트 권한**: 관리자 기능 체크 스크립트의 파일명 정규화 및 권한 안내 보강.

### Changed (변경)
- 전체적인 UI 디자인 가독성 및 시각적 효과 개선 (그라데이션 및 글래스모피즘 강화).
